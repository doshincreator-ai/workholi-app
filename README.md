 # WorkHoli 🌏

  NZ・オーストラリアのワーキングホリデー向け給与管理アプリ

  ## 概要

  ワーホリ中の給与計算（税金・KiwiSaver・Holiday Pay）を自動化し、職場情報をコミュニティで共有するReact Nativeアプリです。

  ## 機能

  - **シフト記録・手取り自動計算** - KiwiSaver / PIR税 / Holiday Pay / スーパーアニュエーション対応
  - **複数の雇用主管理** - 職場ごとに時給・税コードを管理
  - **コミュニティ機能** - みんなの職場情報（時給・難易度・地域・業種）を閲覧・投稿
  - **フレンド機能** - 招待コードで友達と繋がり、勤務先を共有
  - **レポート** - 月別・年別の収支サマリー
  - **NZ・AU 両国対応**

  ## 技術スタック

  | カテゴリ | 技術 |
  |---------|------|
  | フレームワーク | React Native + Expo Router |
  | 状態管理 | Zustand |
  | ローカルDB | expo-sqlite |
  | クラウド | Firebase Firestore / Authentication |
  | ビルド | EAS Build |
  | 言語 | TypeScript |
